# AppointmentBot — запись в барбершоп / тату-салон

Telegram-бот для записи клиентов в барбершоп, тату-салон или любую другую сферу услуг с выбором мастера.

## Возможности

- Выбор мастера и услуги
- Просмотр свободных слотов
- Подтверждение записи
- Автоматическое обновление расписания
- База клиентов

## Стек

- Python 3.10+
- Aiogram 3.x
- SQLite
- FSM

## Установка

```bash
git clone https://github.com/pipupip/appointment-bot
cd appointment-bot
pip install -r requirements.txt
# настройте config.py
python bot.py
```
